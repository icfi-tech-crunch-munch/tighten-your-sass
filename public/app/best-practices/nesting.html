Nesting

Try to avoid nesting as much as possible, utilize classnames to do so.

Do not nest more than two, maybe three levels for tables or similar.

<section class="panel panel-error">
    <header class="panel-header">
        <h1 class="panel-title">Panel Title</h1>
    </header>
    <div class="panel-body">

    </div>
</section>

discouraged:

.panel {
    border: 1px solid #ccc;

    header {
        padding: 15px;
        background-color: lighten(#ccc, 5%);

        h1 {
            font-size: 18px;
        }
    }
}

preferred:

.panel {
    border: 1px solid #ccc;
}

.panel-header {
    padding: 15px;
    background-color: lighten(#ccc, 5%);
}

.panel-title {
    font-size: 18px;
}

.panel-error {
    border: 1px solid #f00;

    .panel-title {
        color: #f00;
    }
}
