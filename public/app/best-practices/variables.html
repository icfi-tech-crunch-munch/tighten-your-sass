Variables

Use variables! Try to reduce 'magic numbers' or literals as much as possible.

Use contextual names.

Like class names, try to avoid literal descriptions.

Discouraged:

$red-border: #f00;
$margin-top-10: 10px;

Preferred:

$border-color-error: #f00;
$margin-top-md: 10px;

Use a consistent naming scheme.

$font-size-base: 14px;
$font-size-lg: $font-size-base * 1.5;
$font-size-sm: $font-size-base * .75;

$border-radius-base: 5px;
$border-radius-lg: $border-radius-base * 2;
$border-radius-sm: $border-radius-base * .75;

Use a _variable.scss file

Place all variables for the site in one file.

Organize the variables in groups that match your partial structure.
