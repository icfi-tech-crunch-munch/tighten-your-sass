Extend

Extending an existing selector creates a comma separated list of selectors for code reuse.

.panel {
    border-width: 1px;
    border-style: solid;
}

.panel-default {
    @extend .panel;

    border-color: #ccc;
}

.panel-error {
    @extend .panel;

    border-color: #f00;
}

Compiles to:

.panel,
.panel-default,
.panel-error {
    border-width: 1px;
    border-style: solid;
}

.panel-default {
    border-color: #ccc;
}

.panel-error {
    border-color: #f00;
}

Placeholder

If you do not want the element you're extending to appear in compiled .css file, replace the selector . with %

%panel {
    border-width: 1px;
    border-style: solid;
}
