Partials & Importing

One advantage of Sass is the ability to organize your CSS into smaller reusable comonents.

Prepending a filename with an underscore, such as _type.scss ensures that that file will not be compmiled as a .css file.

Instead, several partials are added to a .scss file via @include that are compiled to .css

_utilities.scss

.pull-right {
  float: right !important;
}

_type.scss

h1 {
  font-size: 2em;
}

styles.scss

@include 'utilities';
@include 'type';

Note the omission of the .scss extension and the underscore prefix in the include.

The compiled styles.css:

.pull-right {
  float: right !important;
}

h1 {
  font-size: 2em;
}
