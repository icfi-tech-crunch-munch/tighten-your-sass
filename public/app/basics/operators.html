Operators

Number Operators

Arithmatic operations can be applied to variables and literals. +, -, *, /, % are all supported.

There are a few caveats

  • Two literals with a division operator are not divided: font: 10px/5px; will not compile to font: 2px;, but font: (10px/5px); will.
  • Two variables need to be interpolated if you don't want the division operator applied: font: #{$font-size}/#{$line-height}; will compile to font: 10px/5px;
  • Units that aren't compatible will throw an error in operations such as: height: 2em + 10px;

String Operators

Strings can be concatinated with the + operator.

If strings are concatinated with mixed quotes, the quote use of the left element is used.

"Droid" + sans = "Droid Sans"

sans + "-serif" = sans-serif

Boolean Operators

Boolean operations include: and, or, not
