Functions

Sass offers a number of functions, including: color functions, opacity functions, string functions, number functions, list & map functions and a handful of introspection functions

I enourage perusing this list: http://sass-lang.com/documentation/Sass/Script/Functions.html

Custom Functions

If none of the built in Sass functions suite your need, you can create your own function

@function sqrt($number) {
    @return $number * $number;
}

.panel {
    margin: sqrt(5px);
}

Compiles to:

.panel {
    margin: 25px;
}
