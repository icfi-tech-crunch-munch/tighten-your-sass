Nesting

CSS Selectors can be nested to apply a parent selector to several elements

$brand-primary-color: #026475;

#site-header {
    background-color: $brand-primary-color;

    a {
        display: block;
    }
}

Compiles to:

#site-header {
    background-color: $brand-primary-color;
}
    #site-header a {
        display: block;
    }

Parent Reference

Child Selectors can reference their parent element with the & character

$brand-primary-color: #026475;

#site-header {
    background-color: $brand-primary-color;

    a {
        display: block;

        &:hover,
        &:focus {
            background-color: darken($brand-primary-color, 10%);
        }
    }
}

Compiles to:

#site-header {
    background-color: $brand-primary-color;
}
    #site-header a {
        display: block;
    }
    #site-header a:hover,
    #site-header a:focus {
        background-color: #013943;
    }

Nested Imports

If you import a partial into a nested selector, Sass will apply the parent to all of the elements within the partial

_panel.scss

.panel {
    border: 1px solid #ccc;
}

.panel-title {
    font-size: 2em;
}

style.scss

#sidebar {
    @import 'panel';
}

Compiles to

#sidebar .panel {
    border: 1px solid #ccc;
}

#sidebar .panel-title {
    font-size: 2em;
}

Nested Properties

In addition to nesting selectors, Sass can nest properties.

.panel-title {
    font: {
        size: 2em;
        weight: normal;
        family: arial, sans-serif;
    }
}

Compiles to

.panel-title {
    font-size: 2em;
    font-weight: normal;
    font-family: arial, sans-serif;
}
