Variables

Sass allows you to define variables that can be used throughout the stylesheet. This helps reduce "magic numbers" and can speed up global changes.

The $ symbol is used to prepend a variable name.

$brand-primary-color: #026475;
$brand-seconday-color: #a94b08;

#site-header {
    background-color: $brand-primary-color;
}

#site-footer {
    background-color: $brand-secondary-color;
}

Compiles to:

#site-header {
    background-color: #026475;
}

#site-footer {
    background-color: #a94b08;
}

Variable Types

Variables can be any of the following types:

  • numbers or numbers with units 1, 1.2, 12px
  • strings of text with or without quotes "Mike D", 'Ad-Rock', MCA
  • colors red, #ff0000, rgba(255, 0, 0, 1)
  • booleans true, false
  • null null
  • lists separated by spaces or commas 20px 10px 10px 20px -or- Helvetica, Arial, sans-serif
  • maps (weight: 2px, color: #ff0000, style: solid)

Default

By adding !default to the end of the variable declaration, the variable will only be set if it hasn't already been assigned a value

$brand-primary-color: #026475;
$brand-primary-color: #a94b08 !default;
$brand-secondary-color: #a94b08 !default;

#site-header {
    background-color: $brand-primary-color;
}

#site-footer {
    background-color: $brand-secondary-color;
}

Compiles to:

#site-header {
    background-color: #026475;
}

#site-footer {
    background-color: #a94b08;
}

Scope

Variables are scoped to the block they are contained in

$brand-primary-color: #026475;

#site-header {
    background-color: $brand-primary-color;
}

#site-footer {
    $brand-primary-color: #a94b08;
    background-color: $brand-primary-color;
}

Compiles to:

#site-header {
    background-color: #026475;
}

#site-footer {
    background-color: #a94b08;
}

Interpolation

Variables can be used in selector and property names via interpolation syntax #{}

$id: #site-header;
$property: background;
$brand-primary-color: #026475;

header#{$id} {
    #{$property}-color: $brand-primary-color;
}

Compiles to:

header#site-header {
    background-color: #026475;
}
