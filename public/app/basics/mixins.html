Mixins

Sass mixins allow you to easily reuse blocks of properties, such as vendor prefixes.

@mixin size ($width, $height) {
    width: $width;
    height: $height;
}

.thumbnail {
    @include size(160px, 90px);
}

Compiles to:

.thumbnail {
    width: 160px;
    height: 90px;
}

Argument Default Values

Mixin arguments can accept default values.

@mixin size ($width: auto, $height: auto) {
    width: $width;
    height: $height;
}

.thumbnail {
    @include size(160px);
}

Compiles to:

.thumbnail {
    width: 160px;
    height: auto;
}

Arguments passed as a map

Mixins can accept a map for arguments.

@mixin size ($width: auto, $height: auto) {
    width: $width;
    height: $height;
}

.thumbnail {
    @include size($height: 150px);
}

Compiles to:

.thumbnail {
    width: auto;
    height: 150px;
}
