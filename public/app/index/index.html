Tighten Your Sass

If you have Node.js installed and would like to follow along, run the following commands in your local repository directory:

git remote add upstream https://github.com/icfi-tech-crunch-munch/tighten-your-sass
git pull upstream master
npm install
bower install
node server.js

Then visit http://localhost:3030
